Gnome

How to install a GNOME extension starting from the zip archive

  1. Get the extension’s uuid:
    $ unzip -c [zip-file] metadata.json | grep uuid | cut -d \" -f4
  2. Create the destination directory:
    $ mkdir -p ~/.local/share/gnome-shell/extensions/[extension-uuid]
  3. Unzip the GNOME extension:
    $ unzip -q [zip-file] -d ~/.local/share/gnome-shell/extensions/[extension-uuid]/
  4. Enable GNOME extension:
    $ gnome-extensions enable [extension-uuid]
