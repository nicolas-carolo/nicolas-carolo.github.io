Pandoc

Requirements

Arch

  1. Install texlive-most: 
    $ sudo pacman texlive-most
  2. Install pandoc: 
    $ sudo pacman -S pandoc

Fedora

  1. Install texlive-scheme-full: 
    $ sudo dnf install texlive-scheme-full
  2. Install pandoc: 
    $ sudo dnf install pandoc

Commands

  • Convert a markdown file into a PDF file: 
    $ pandoc [markdown file]  -s -o [PDF file]
