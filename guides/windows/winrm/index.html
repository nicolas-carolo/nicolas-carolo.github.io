Windows Remote Management

Enable Windows Remote Management (WinRM) in Windows 10

  1. WIN + R and launch gpedit.msc
  2. Go to: Local Computer Policy -> Computer Configuration -> Windows Settings -> Security Settings -> Windows Defender Firewall with Advanced Security -> Windows Defender Firewall with Advanced Security - Local Group Policy Object -> Inbound Rules
  3. Right click on Inbound Rules and click on New Rule…
  4. Choose Predefined and select Windows Remote Management from the drop-down list
  5. Press Next and then Next again
  6. Choose Allow the connection and click Finish
  7. WIN + R and launch services.msc:
  8. Go to Windows Remote Management, right click, click on Start
  9. Right click on Windows Remote Management, select Properties and choose Automatic option in the drop-down menu for selecting the Startup type
  10. Click on Apply and then on OK
